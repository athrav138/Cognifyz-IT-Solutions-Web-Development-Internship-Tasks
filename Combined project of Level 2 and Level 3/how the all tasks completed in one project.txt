# Cognifyz Web Development Internship Task Completion Report

## Project Name

Professional Personal Portfolio Website

## Project Overview

This project is a fully responsive personal portfolio website designed to showcase my profile, technical skills, projects, certifications, achievements, and contact information.

Instead of creating separate small websites for each task, I integrated all the required functionalities from Level 2 and Level 3 into a single real-world web application. This demonstrates practical web development skills and follows modern industry standards.

---

# Level 2 Task 1: Front-end Frameworks

## Requirement:

Create a responsive card component using a front-end framework and implement a responsive grid layout.

## Implementation in Portfolio:

* Bootstrap 5 framework is used throughout the website.
* The Projects section uses responsive Bootstrap card components containing:

  * Project image
  * Project title
  * Project description
  * Technology stack
  * GitHub and live demo links
* The Skills, Projects, Certifications, and Achievements sections are organized using Bootstrap Grid System.
* The layout automatically adjusts based on screen size:

  * Desktop: Multiple columns
  * Tablet: Two columns
  * Mobile: Single column

**Task Status: Completed ✔**

---

# Level 2 Task 2: Responsive Design

## Requirement:

Convert a webpage into a responsive design using CSS media queries and create a mobile-friendly hamburger navigation menu.

## Implementation in Portfolio:

* The complete website follows responsive design principles.
* CSS media queries are used to adjust layouts, typography, images, spacing, and components for different devices.
* A responsive navigation bar is implemented.
* On larger screens, the navigation links are displayed horizontally.
* On smaller screens, the navigation collapses into a hamburger menu for a better mobile experience.
* The website is optimized for desktop, tablet, and mobile devices.

**Task Status: Completed ✔**

---

# Level 3 Task 1: Image Manipulation

## Requirement:

Create an image gallery with clickable thumbnails and implement an automatic image slideshow.

## Implementation in Portfolio:

* A certificate and project showcase gallery is included.
* Images are displayed as responsive thumbnails.
* Clicking on an image opens an enlarged preview using a modal/lightbox effect.
* A slideshow/carousel is implemented to automatically display featured projects, certifications, or achievements.
* The slideshow includes smooth transitions and works across different screen sizes.

**Task Status: Completed ✔**
