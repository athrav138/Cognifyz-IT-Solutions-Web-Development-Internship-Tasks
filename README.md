### Combined Project: WebScraper Pro

The combined software project is located at:

`1 Month Software Development Internship - Atharv MIlind Suryavanshi/All Task Combined Project/Web Scrapper v1.0.0/`

WebScraper Pro is a desktop application built with Python and PyQt6. It provides a modern interface for scraping website data and managing saved results.

#### Features

- Modern dark-themed PyQt6 desktop interface.
- Website scraping for page titles, headings, links, images, and metadata.
- SQLite database storage for scrape history.
- Dashboard view with scraping statistics and recent activity.
- History page for browsing, inspecting, searching, and deleting previous scrapes.
- Settings page for timeout, user-agent, and parsing limits.
- Export support for TXT, JSON, CSV, and PDF-style reports.
- Build script and installer configuration for packaging the application.

#### Run the Desktop App

```bash
cd "1 Month Software Development Internship - Atharv MIlind Suryavanshi/All Task Combined Project/Web Scrapper v1.0.0"
pip install -r requirements.txt
python main.py
```

The application creates local runtime files such as the SQLite database and settings file automatically when needed.

#### Build the Desktop App

```bash
pip install pyinstaller
python build.py
```

The packaged output is generated in the project `dist/` folder.
